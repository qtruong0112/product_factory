## VỊ TRÍ THƯ MỤC (quan trọng)
Gốc dự án nằm ở **`product-factory/`** (thư mục lồng), KHÔNG ở gốc repo:
- `product-factory/backend/`   → Java 21 + Spring Boot 3.3.4 + Maven, **Clean Architecture layer-first** (Giai đoạn 27): package `com.f88.productfactory.{domain.model.<feature>|infrastructure.persistence.<feature>|application.service.<feature>|application.common|application.dto.<feature>|infrastructure.config|presentation.controller.<feature>|presentation.common}` (**Giai đoạn 82**: Spring Data repository interface dời từ `domain.repository` sang `infrastructure.persistence` — domain giờ CHỈ còn entity thuần, đúng Dependency Rule của Clean Architecture: domain không được phụ thuộc framework/chi tiết kỹ thuật).
